# word-map-api
Takes a word array and returns the synonyms/antonyms related to each.

For example:
```
send:
{
    "words": ["Beautiful", "Cool", "collected"]
}
```
and receive:
```
{
    "beautiful": {
        "antonyms": [
            "ugly"
        ],
        "synonyms": [
            "beautiful"
        ]
    },
    "collected": {
        "antonyms": [
            "spread",
            "uncollected",
            "ungathered"
